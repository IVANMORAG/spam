Introducción al Proyecto

Detección de Intrusiones de Red con Machine Learning

Este proyecto implementa técnicas de Machine Learning para la detección de intrusiones en redes de computadoras utilizando el dataset NSL-KDD.

Dataset NSL-KDD

Descripción

ISCX NSL-KDD dataset 2009 es un conjunto de datos propuesto para resolver algunos de los problemas inherentes del dataset KDD'99.

Aunque esta nueva versión aún sufre de algunos problemas, debido a la falta de conjuntos de datos públicos para IDS basados en red, sigue siendo un benchmark efectivo para comparar diferentes métodos de detección de intrusiones.

Ventajas del NSL-KDD
  • No incluye registros redundantes en el conjunto de entrenamiento
  • No hay registros duplicados en los conjuntos de prueba
  • El número de registros seleccionados de cada nivel de dificultad es inversamente proporcional al porcentaje del dataset original
  • El número de registros es razonable para ejecutar experimentos en el conjunto completo

Archivos del Dataset

Archivo Descripción

KDDTest+.ARFF Conjunto completo de prueba
KDDTest-21.ARFF Subconjunto sin registros de nivel de dificultad 21
